# Quiz-App-Beta

> Desarrollo de un Quiz consumiendo una Fake API con json-server

# Usage

## Run json Fake API (Command Prompt)

```
> cd quizApp

> quizApp> json-server db.json


```
## Run App

```
> cd quizApp

> quizApp> npm install

> quizApp> react-native run-android

```
